## ğŸ†˜ Troubleshooting

### App won't install?
- Use Safari (not Chrome) on iPhone/iPad
- GitHub Pages must be enabled
- Wait 2-3 minutes after enabling Pages

### Handwriting not working?
- Check if SignaturePad loaded (see browser console)
- Try refreshing the page
- Works best with Apple Pencil

### Data not saving?
- Check browser storage settings
- Clear cache and reload
- Export data regularly as backup

### Icons not showing?
- Wait for GitHub Pages to fully deploy
- Hard refresh: Hold Shift + Click Reload
- Check manifest.json paths
